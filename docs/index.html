Introdução e Orientações

Este painel apresenta os principais resultados da análise espacial dos estabelecimentos de saúde de Concórdia/SC. Foram processados dados do CNES, classificados os tipos de unidades e gerados mapas interativos para facilitar a compreensão da cobertura e distribuição dos serviços de saúde no município.

Como utilizar os mapas:

  • Navegue pelos mapas clicando nos botões "Abrir Mapa" de cada card.
  • Explore os marcadores para visualizar informações detalhadas de cada unidade.
  • Utilize os controles de camadas para alternar entre diferentes tipos de estabelecimentos e áreas de influência.
  • Consulte o dashboard para visualizar gráficos, estatísticas e síntese dos resultados.
  • Todos os mapas são responsivos e podem ser acessados em dispositivos móveis.

Os mapas e dashboards foram desenvolvidos seguindo padrões de qualidade geoespacial, garantindo precisão, acessibilidade e clareza visual para apoio ao planejamento territorial e à gestão pública.
